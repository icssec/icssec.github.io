前期准备

注册

​ 简要说明一下怎么进行注册，打开链接 https://www.hackthebox.eu/invite 在console输入 makeInviteCode()

​ 可以得到一个base64加密的数据

SW4gb3JkZXIgdG8gZ2VuZXJhdGUgdGhlIGludml0ZSBjb2RlLCBtYWtlIGEgUE9TVCByZXF1ZXN0IHRvIC9hcGkvaW52aXRlL2dlbmVyYXRl

解码后提示信息为

​ In order to generate the invite code, make a POST request to /api/invite/generate

我这里通过bp进行发包，获取key，获取到的code字段还需要进行base64解密,解密后的验证码就直接可以进行注册了。

使用

官方会提供openvpn链接，因为我这里使用的是mac下载的是Tunnelblick。

进行下载openVPN链接包，通过软件进行拨号链接
