前期准备

注册

​ 简要说明一下怎么进行注册，打开链接 https://www.hackthebox.eu/invite 在console输入 makeInviteCode()

​ 可以得到一个base64加密的数据

SW4gb3JkZXIgdG8gZ2VuZXJhdGUgdGhlIGludml0ZSBjb2RlLCBtYWtlIGEgUE9TVCByZXF1ZXN0IHRvIC9hcGkvaW52aXRlL2dlbmVyYXRl

解码后提示信息为

​ In order to generate the invite code, make a POST request to /api/invite/generate

我这里通过bp进行发包，获取key，获取到的code字段还需要进行base64解密,解密后的验证码就直接可以进行注册了。

使用

官方会提供openvpn链接配置文件，因为我这里使用的是mac下载的是Tunnelblick。

进行下载openVPN链接包，通过软件进行拨号链接

Cartographer

介绍：一些地下黑客正在开发新的命令和控制服务器。您可以闯入看看他们在做什么吗？

在拿到网站的时候尝试了一下admin和top100的弱口令，失败后，感觉可能是万能密码登录，直接通过字典进行爆破尝试,成功admin’or’a’=’a

登陆成功后网页重定向到了http://docker.hackthebox.eu:30642/panel.php?info=home

http://docker.hackthebox.eu:30642/panel.php?info=home’ 网页提示not fount！

怀疑可能是文件包含，尝试将home修改成flag。成功获取flag

Emdee five for life

介绍:您可以加密得足够快吗？

这里不管怎么提交都会提示Too slow！

想起了当时bugku 里面的一个题，直接尝试编写python脚本进行提交。

直接使用burp生成简单的修改一下，OK

测试脚本地址：

https://github.com/icssec/HacktheboxExp/blob/master/hackbox-Emdee%20five%20for%20life.py

Fuzzy

介绍:我们已经获得了一些我们认为与目标内部网络相连的基础架构的访问权。我们需要您帮助获取他们当前正在开发的网站的管理员密码。

dirmap一波发现一个http://docker.hackthebox.eu:30738/api/ 但是页面是空的，蒙了~

找了一下网上的人解题思路，蒙蔽了，这个大佬是用的dirbuster,这api下面有一个action.php

心思尝试一下，时间太长了。算了吧。访问页面后提示Error: Parameter not set

还是通过dirbuster进行测试。

fuzz到参数后进行访问页面提示Error: Account ID not found

直接通过burp进行测试一下ID字段的参数，通过burp的number进行测试

payload确定为20，直接查看response获得flag

Lernaean

  1. 简单使用dirmap跑了一波 无果

  2. 标题网站页面的提示’请不要尝试猜测我的密码！’ 那就必须得尝试一下啊。

  3. 上来使用sqlitop进行尝试，失败了~不能使用万能密码登录

  4. 那就尝试一下top的字典密码吧

    我这里使用的是

    https://github.com/TheKingOfDuck/fuzzDicts

  5. 发现密码是:leonardo 直接获得flag
